============================================
   Layout Switch — המרת שפה בלחיצה
============================================

הבעיה: כתבת טקסט שלם ואז גילית שהיית בשפה הלא נכונה?
הפתרון: סמן את הטקסט → לחץ F8 → מתוקן!

דוגמה:
   akuo  →  שלום
   שלום  →  akuo

============================================
   התקנה (פעם אחת)
============================================

1. פתח את קובץ הZIP
2. שים את 2 הקבצים באותה תיקייה
3. לחץ פעמיים על "LayoutSwitch-Installer.bat"
4. אם Windows שואל "האם לאפשר?" → לחץ "כן"
5. חכה שההתקנה תסתיים
6. זהו! F8 עובד מעכשיו

============================================
   שימוש
============================================

1. סמן טקסט שהוקלד בשפה הלא נכונה
2. לחץ F8
3. הטקסט מתהפך לשפה הנכונה

עובד בכל מקום: Word, Chrome, WhatsApp, VS Code,
Outlook, Excel, Notepad — כל תוכנה.

עולה אוטומטית עם Windows — אין צורך להפעיל שוב.

============================================
   הסרה
============================================

למחיקה: מחק את הקובץ מתיקיית Startup:
%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\LayoutSwitch.ahk
